# QUALITY_SCORE

本文件定义质量评分维度与门禁规则，供智能体与人类一致执行。

## 评分维度（建议权重）
- 正确性（30%）：功能与需求匹配
- 可靠性（20%）：失败恢复、SLO 达标
- 可维护性（15%）：结构清晰、可扩展
- 可测试性（15%）：测试覆盖与稳定性
- 可观测性（10%）：日志/指标/追踪可读
- 安全性（10%）：权限、审计、合规

## 门禁规则（最小门槛）
- 总分 >= 80
- 正确性 >= 25
- 可靠性 >= 15
- 安全性 >= 8
- 任何 P0 级缺陷必须阻断发布

## 评分证据
每次评分应附带以下证据：
- 测试报告与覆盖率摘要
- 关键路径运行日志
- 主要变更列表与审计记录
